# Dev Daily (CS Job Search Tracker)
CS majors have it rough nowadays especially in this market. This tool can help with staying on track regarding job hunting by allowing you to track your activities every day.
## To-Do
- Allow the user to select their goals regarding:
1. Number of job applications filled
2. LeetCode problems solved
3. Time spent on personal projects
- Allow the user to fill out this information on a form.
- They can then submit this "entry" once per day.
- Display the entries as cards and color them green, yellow, or red depending on if they hit their goals or not.
